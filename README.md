## üöÄ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- MongoDB account/local instance

### 1. Clone the repository
```bash
git clone <repository-url>
cd finance-tracker
```

### 2. Setup Server
```bash
cd Server
npm install
# Create a .env file (see Server/README.md for details)
npm run dev
```

### 3. Setup Client
```bash
cd Client
npm install
npm run dev
```
